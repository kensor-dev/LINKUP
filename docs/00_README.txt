ЧТО МЫ СТРОИМ
--------------
B2B SaaS для малого бизнеса с собственными курьерами.
GPS-трекинг + CRM. Одна система, одна база данных.

Целевой клиент: ресторан, цветочный, аптека — любой бизнес
с 1–20 своими курьерами, который сейчас управляет ими через WhatsApp.


ЧТО ВХОДИТ В ПРОДУКТ
---------------------
  GPS-ТРЕКИНГ
    Приложение курьера (мобильное)
    Карта со всеми курьерами в реальном времени (дашборд)
    Живая ссылка на трекинг для клиента (без приложения)
    Автоматические уведомления клиенту (SMS / WhatsApp)
    Управление заказами (диспетчерский модуль)
    Аналитика курьеров (рейтинг, среднее время, опоздания)
    Управление сменами курьеров

  CRM
    Автоматический профиль клиента (создаётся при каждом заказе)
    История заказов клиента
    Сегменты клиентов (новый / постоянный / засыпает / потерян)
    Теги и заметки к клиенту
    Аналитика CRM (retention, LTV)
    Визуальный конструктор автосценариев
    Задачи для менеджеров

  МАРКЕТПЛЕЙС — Фаза 3, далёкое будущее (не строим сейчас)
